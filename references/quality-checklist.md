# 出题质量检查清单

## 一、答案正确性（P0 必检）

- [ ] 每题答案经二次验证，确认准确无误
- [ ] 选择题有且仅有一个最佳答案（多选题除外）
- [ ] 判断题的命题有明确对错，无歧义
- [ ] 填空题答案唯一或属于有限可接受集合
- [ ] 计算题数值结果精确，分步过程无逻辑错误
- [ ] 简答/论述题答案要点完整，得分点清晰

## 二、题干质量（P0 必检）

- [ ] 题干包含解答所需的全部已知条件
- [ ] 题干表述无歧义，理解方向唯一
- [ ] 题干无错别字、语法错误、标点错误
- [ ] 否定式题干已加粗标注否定词
- [ ] 题干长度适中，不过长（>100字需分行）也不过短（<10字需补充）

## 三、选项质量（P1 应检）

- [ ] 每题至少3个干扰项（选择题）
- [ ] 干扰项有合理错误来源，非明显荒谬选项
- [ ] 选项长度大致相当，正确答案非最长/最短
- [ ] 选项间无逻辑包含关系
- [ ] "以上都对/都不对"占比<5%
- [ ] 多选题错误选项具备部分正确性（"看起来对"）

## 四、认知层次标注（P1 应检）

- [ ] 每题标注Bloom层次（L1-L6）
- [ ] 标注层次与题型匹配（如判断题不标L5-L6）
- [ ] 同卷Bloom层次分布符合预设难度比例
- [ ] 高分值题对应高认知层次

## 五、知识点覆盖（P1 应检）

- [ ] 试卷覆盖第二步识别的全部核心概念
- [ ] 无知识点过度集中（单知识点题数<总题数30%）
- [ ] 无知识点完全重复的题目
- [ ] 重点知识点有多个角度/层次的考查

## 六、难度分布（P2 建议检）

- [ ] 基础/进阶/挑战比例符合预设（默认40/40/20）
- [ ] 同难度档内题目实际难度均衡
- [ ] 试卷整体难度与目标学段匹配
- [ ] 无明显偏题、怪题、超纲题

## 七、解析质量（P1 应检）

- [ ] 每题附详细解析，非仅给出答案
- [ ] 解析说明"为什么选这个"和"为什么不选其他"
- [ ] 计算题解析含完整求解过程
- [ ] 解析标注考点和常见误区
- [ ] 解析语言与目标学段匹配（小学用通俗语言，大学用专业术语）

## 八、格式规范（P2 建议检）

- [ ] 题号连续无跳号
- [ ] 题型分组，同类型题目排列在一起
- [ ] 分值标注清晰
- [ ] 选项缩进对齐
- [ ] 数学公式/化学方程式格式正确

## 质量问题修正流程

1. 发现P0问题 → 立即修正，重新校验
2. 发现P1问题 → 修正后继续，同批次复查同类问题
3. 发现P2问题 → 批量记录，统一修正
4. 修正后需对修改项重新跑一遍P0检查
