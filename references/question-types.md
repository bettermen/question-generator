# 题型设计规范

## 1. 单选题

### 结构要求
- 题干：完整陈述或疑问句，包含全部已知条件
- 选项：4个（A/B/C/D），1个正确答案
- 干扰项：3个，每个有合理错误来源

### 设计要点
- 避免否定式题干（"下列哪个不是..."），必须使用时加粗"不"
- 选项长度大致相当，避免正确答案明显最长
- 避免选项间有逻辑包含关系（如A正确则B必然正确）
- "以上都对"/"以上都不对"控制在5%以内

### 范例
```
Q: 某物体做匀加速直线运动，初速度为2m/s，加速度为3m/s²，第3秒末的速度为？
A. 8 m/s
B. 9 m/s
C. 11 m/s
D. 15 m/s
答案：C
解析：v = v₀ + at = 2 + 3×3 = 11 m/s。A选项误用v = v₀ + at/2；B选项误用v = at；D选项误用v = v₀ + at²。
```

## 2. 多选题

### 结构要求
- 题干：明确说明"多选"或"至少两个选项正确"
- 选项：4-5个，2-4个正确答案
- 干扰项：至少1个，具备部分正确性

### 设计要点
- 正确答案数量以2-3个为主，4个以上的全选题应控制在10%以内
- 每个选项独立可判，避免选项间存在依赖关系
- 错误选项应"看起来对"，考查学生对细节的把握

### 范例
```
Q: 下列关于光合作用的叙述，正确的有（多选）？
A. 光反应在类囊体薄膜上进行
B. 暗反应需要ATP和NADPH
C. 光合作用释放的氧气来自水的分解
D. 暗反应只在黑暗中进行
答案：ABC
解析：A正确，光反应场所是类囊体薄膜；B正确，暗反应利用光反应产生的ATP和NADPH还原C3；C正确，光反应中水的光解释放氧气；D错误，暗反应在有光无光条件下均可进行，但需要光反应提供的产物。
```

## 3. 判断题

### 结构要求
- 陈述句：一个完整的判断命题
- 答案：对/错
- 解析：说明对或错的原因

### 设计要点
- 每题只包含一个判断点，避免半对半错
- 避免使用绝对化词语（"总是""一定""绝不"）
- 错误题应有明确的错误位置，而非整体编造

### 范例
```
Q: 在DNA分子中，腺嘌呤(A)总是与胸腺嘧啶(T)配对，鸟嘌呤(G)总是与胞嘧啶(C)配对。
答案：对
解析：这是DNA碱基互补配对原则，A-T通过两个氢键配对，G-C通过三个氢键配对。
```

## 4. 填空题

### 结构要求
- 题干：留空处用____标注
- 答案：唯一或有限集合
- 每空1-3分

### 设计要点
- 每空只填一个确定答案，避免开放性填空
- 空格长度一致，不暗示答案长度
- 关键术语填空优于数值填空（数值用计算题考查）

### 范例
```
Q: 细胞呼吸的第一阶段是____，发生在细胞质的____中。
答案：糖酵解；基质
解析：细胞呼吸第一阶段为糖酵解，将葡萄糖分解为两分子丙酮酸，发生在细胞质基质中，不需要氧气参与。
```

## 5. 简答题

### 结构要求
- 题干：明确提问，限定回答范围
- 答案：要点式，3-5个得分点
- 分值：5-8分

### 设计要点
- 题干用"简述""说明""列出"限定深度
- 答案预设得分点，每点1-2分
- 避免开放性过大导致评分主观

### 范例
```
Q: 简述酶催化的三个特点。（6分）
答案：
1. 高效性（2分）：酶的催化效率远高于无机催化剂，可加快反应速率10⁷~10¹³倍。
2. 专一性（2分）：每种酶只能催化一种或一类底物的反应。
3. 作用条件温和（2分）：酶在常温常压、适宜pH下起作用，高温强酸强碱会失活。
```

## 6. 论述题

### 结构要求
- 题干：给出情境或命题，要求分析论证
- 答案：结构化论述，含论点/论据/结论
- 分值：10-15分

### 设计要点
- 题干要求明确（"从X角度分析""比较A和B的异同"）
- 答案预设评分维度（论点/论据/逻辑/结论各占分）
- 考查高阶思维（L4-L6），不用于简单记忆

### 范例
```
Q: 论述 globalization 对发展中国家制造业的双重影响。（12分）
答案要点：
1. 积极影响（4分）：技术转移与产业升级、融入全球供应链、创造就业机会
2. 消极影响（4分）：低端锁定风险、环境标准降低、本土企业竞争压力
3. 辩证分析（2分）：影响程度取决于国家产业政策、教育水平、基础设施
4. 结论（2分）：需在开放与保护间寻找平衡，主动提升产业链位置
```

## 7. 计算题

### 结构要求
- 题干：已知条件 + 求解目标
- 答案：分步求解过程 + 最终结果
- 分值：8-12分

### 设计要点
- 已知条件完整，数值合理（避免过于复杂的计算）
- 答案分步给分（公式/代入/计算/结果各占分）
- 考查核心公式理解，非计算技巧

### 范例
```
Q: 质量为2kg的物体从高5m处自由下落（g=10m/s²），求：
(1) 物体落地时的速度（4分）
(2) 物体落地时的动能（4分）
答案：
(1) v² = 2gh = 2×10×5 = 100，v = 10m/s（公式2分，计算2分）
(2) Ek = ½mv² = ½×2×100 = 100J（公式2分，计算2分）
```

## 8. 应用题

### 结构要求
- 题干：真实或拟真情境 + 待解决的问题
- 答案：问题分析 + 解决方案 + 过程计算
- 分值：8-12分

### 设计要点
- 情境真实可信，数据合理
- 需要将知识迁移到新情境
- 答案允许多种合理解法

### 范例
```
Q: 某家庭月收入15000元，计划储蓄收入的20%，其余用于生活开支。已知固定开支（房租+水电）6000元/月，弹性开支（餐饮+交通+娱乐）需控制在多少范围内？请计算并给出建议。（10分）
答案：
储蓄 = 15000×20% = 3000元
可支配 = 15000-3000 = 12000元
弹性开支上限 = 12000-6000 = 6000元
建议：弹性开支控制在4000-6000元，留出应急空间。（分析4分+计算4分+建议2分）
```

## 9. 匹配题

### 结构要求
- 左列：概念/术语（4-6项）
- 右列：定义/描述（4-6项，可打乱顺序）
- 答案：对应关系

### 设计要点
- 左列同质（同类型概念），右列同质
- 右列项数可多于左列，增加难度
- 每个匹配项独立可判

### 范例
```
Q: 将左侧生物分子与右侧对应功能进行匹配。
左列：①DNA ②mRNA ③tRNA ④核糖体
右列：A.携带氨基酸 B.蛋白质合成场所 C.遗传信息载体 D.传递遗传信息
答案：①-C ②-D ③-A ④-B
```

## 10. 案例分析题

### 结构要求
- 案例材料：200-500字真实或拟真场景
- 问题：2-4个子问题，由浅入深
- 答案：结合案例理论分析
- 分值：15-20分

### 设计要点
- 案例材料信息充分，支持所有子问题
- 子问题层次递进（识别→分析→评价→建议）
- 答案需引用案例细节，不能纯理论套话

### 范例
```
案例：某科技公司A推出一款AI教育产品，上线3个月后用户留存率从60%降至15%。调查发现：产品功能完善，但教师反映"不会用"、学生反映"不好玩"。
问题：
(1) 分析用户留存率下降的两个主要原因。（5分）
(2) 从产品角度提出3条改进建议。（5分）
(3) 设计一个验证改进效果的方案。（5分）
答案要点：
(1) 教师使用门槛高（缺乏引导/培训）；学生动机不足（缺乏趣味性设计）
(2) 增加新手引导/教师培训模块；加入游戏化学习元素；建立用户反馈闭环
(3) A/B测试方案：实验组用新版、对照组用旧版，对比7日/30日留存率，样本量>1000
```
